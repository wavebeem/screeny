pixel
the smallest addressable unit of color on a screen.
dip
the smallest addressable integer unit of color in software. apple originally defined a point on a retina display as a 2x2 grid of hardware pixels. this allows "half pixel" values to be drawn to the screen without color blending. android prefers the term "dip", an acronym for "device indepent pixel". i will use the term "dip" to avoid confusion with the unrelated "point" used in font sizes.
dip size
percentage difference in dips per inch between screens. a larger number means that an object drawn at the same size in software will be that much larger physically on the new screen.
dip detail
how many hardware pixels back each dip (software pixel). retina typically means each dip is a 2x2 square of hardware pixels.
font size
a 16 dip font size looks like this many dips on the new display.
pixels per inch
the number of pixels per inch on the screen.
dips per inch
the number of dips per inch on the screen. smaller numbers make objects look bigger.
